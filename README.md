# Dynamic Parking Pricing System – Web UI

This project is a simple, modern React-based web interface for displaying dynamic parking prices generated by a serverless Azure Function pricing engine.

It fetches pricing data from an Azure Function API, processes the JSON response, and presents it in a clean, responsive dark-themed dashboard with search, sorting, and pagination features.

---

## System Architecture Overview

Forecast Model (Local / Research Environment)  
→ Azure Blob Storage (forecast CSV)  
→ Azure Function (Serverless Pricing Engine API)  
→ React Web UI (Static Frontend)  

The frontend is completely static and can be hosted for free using:

- Azure Static Web Apps
- Vercel
- GitHub Pages

---

## Features

- Dark professional UI
- Mobile responsive layout
- Search by parking_id
- Top 10 highest dynamic prices
- Paginated data table
- Sticky table header
- Page size selector
- API refresh button
- Error handling
- Works with both:
  - Array JSON response
  - { meta, data } structured JSON

---

## Tech Stack

Frontend:

Backend:
